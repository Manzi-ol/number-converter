# Number Converter

> Live conversion between **Binary, Octal, Decimal, and Hexadecimal** with step-by-step explanations.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)

---

## Features

- Live simultaneous conversion across all 4 bases as you type
- Step-by-step explanation of how the conversion was calculated
- Input validation — highlights invalid characters per base
- Copy-to-clipboard button for each result
- Supports large numbers
- Dark/light mode
- No dependencies — open `index.html` directly

---

## Run

```bash
git clone https://github.com/Manzi-ol/number-converter
# Open index.html in your browser
```

---

## Supported Bases

| Base | Name | Characters |
|------|------|------------|
| 2 | Binary | 0, 1 |
| 8 | Octal | 0–7 |
| 10 | Decimal | 0–9 |
| 16 | Hexadecimal | 0–9, A–F |

---

## Tech Stack

- **Vanilla JavaScript** — Conversion logic
- **HTML5 / CSS3** — UI and layout
