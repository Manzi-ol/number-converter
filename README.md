# Number System Converter

**#17 of Manzi's 100 GitHub Projects Roadmap**

A live number system converter that translates between Binary, Octal, Decimal, and Hexadecimal — with step-by-step explanations and a powers-of-2 reference table.

## Features

- **4 bases supported:** Binary (base 2), Octal (base 8), Decimal (base 10), Hexadecimal (base 16)
- Type in **any** field and all others update instantly
- Input validation — each field only accepts characters valid for its base
- Color-coded panels: green (decimal), gold (binary), blue (octal), purple (hex)
- **Step-by-step explanation** showing the full division-by-base process
- Positional notation breakdown for binary values
- **Powers of 2 quick-reference table** (2⁰ through 2¹⁵)
- Error highlighting on invalid input

## Usage

Open `index.html` in your browser — no build step needed.
